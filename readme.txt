With Beautiful Social Widget, you can link to your social media profile through Sidebar Widget. 

== Description ==
This Plugin adds a widget functinality to your blog and you can place this widget in the Sidebar and link to your Social Media profile like Facebook and Twitter. 

Currently this widget supports 11 Social Media including Rss and Email me option. This Plugin is easy to setup, you can configure the urls from Widgets option page. Also you have to include the full url including 'http://'.

This Plugin supports the following Social Media: Facebook, Twitter, Google Plus, Linked In, Pinterest, YouTube, Instagram, Tumblr, Flickr, Rss and Email Me.

== Installation ==

1. Download this beautiful-social-widget.zip file to your computer or Optionally you can install by searching from Add New under Plugins tab and Search Beautiful Social Widget and click on Install .
2. Go to Plugins and click Add New to upload and activate the plugin.
3. Go to Appearance >> Widgets and add Beautiful Social Widget and set up the url and save.
